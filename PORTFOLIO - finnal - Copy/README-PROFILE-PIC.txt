PROFILE PICTURE INSTRUCTIONS
============================

To add your profile picture:

1. Save your pixel art profile picture as "profile-pic.png"
2. Place it in the same folder as index.html
3. The image will automatically be displayed with a pixel-art frame
4. Recommended size: 100x100 pixels or larger (will be scaled to fit)
5. Supported formats: PNG, JPG, GIF

If the image doesn't load, the emoji avatar will be shown as a fallback.
